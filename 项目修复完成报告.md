## 技术实现详情

### UUID兼容性模块 (`src/core/uuid_compat.py`)
```python
class GUID(TypeDecorator):
    """跨数据库的UUID类型实现"""
    - 支持PostgreSQL: 使用原生UUID类型
    - 支持SQLite: 存储为16字节二进制
    - 支持其他数据库: 存储为32字符十六进制字符串
```

### 数据库配置自动检测

if "sqlite" in database_url:

    engine_config = {
        "poolclass": NullPool,
        "connect_args": {"check_same_thread": False}

else:
    # PostgreSQL配置
