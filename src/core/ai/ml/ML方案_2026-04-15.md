## 一、定位与目标

### 当前状态
- 已有：`AIScoreEngine`（规则驱动评分）、23指标战法、4套选股策略
- 缺：ML模型驱动的**概率预测**能力

### 本模块目标
在 `src/core/ai/ml/` 下新增 ML 层，将技术指标数据转化为：
1. **N日涨跌概率预测**（次日、5日、20日）
2. **量化信号评分**（机器学习增强AIScoreEngine）
3. **风险量化**（VaR / 最大回撤预测）
4. **模型可解释性**（SHAP特征重要性）

## 四、模型设计

### 4.1 涨跌方向预测模型（PriceDirectionModel）

```python
class PriceDirectionModel:
    """
    XGBoost 分类器：预测N日后涨跌方向
    目标变量：次日收盘 / 5日收盘 / 20日收盘
    """
    model_type = "XGBClassifier"
    objectives = {
        "next_day":  "binary:logistic",   # 涨=1，跌=0
        "next_5d":   "binary:logistic",
        "next_20d":  "binary:logistic",
    }
```

**模型配置：**
```python
params = {
    "objective": "binary:logistic",
    "eval_metric": "auc",
    "max_depth": 6,
    "learning_rate": 0.05,
    "subsample": 0.8,
    "colsample_bytree": 0.8,
    "scale_pos_weight": 1.2,   # A股跌多涨少，略微加权上涨
    "tree_method": "hist",
}
```

### 4.2 风险量化模型（RiskQuantModel）

```python
class RiskQuantModel:
    """
    LightGBM 回归器：预测未来N日最大回撤和VaR
    """
    model_type = "LGBMRegressor"
```

## 五、训练流程

```
数据获取 (AkShare)
    ↓
特征工程 (100+特征)
    ↓
标注构建（未来1/5/20日涨跌标签）
    ↓
训练/验证/测试集划分（时序划分，不打乱）
    ↓
XGBoost + LightGBM 双模型训练
    ↓
SHAP 可解释性分析
    ↓
模型持久化（joblib + MLflow）
    ↓
实时推理 API
```

**数据划分原则（重要）：**
- 不使用随机打乱（避免未来数据泄露）
- 采用滚动窗口回测：Train=2018-2022，Val=2023，Test=2024-2025
- 每月末重训练（增量训练支持）

## 六、评估指标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| AUC-ROC | ≥ 0.60 | 高于随机（0.5） |
| Accuracy | ≥ 52% | 至少超过随机5% |
| Precision | ≥ 55% | 买入信号的准确性 |
| Sharpe Ratio | ≥ 1.0 | 回测年化夏普比率 |
| Max Drawdown | ≤ 15% | 最大回撤控制 |

## 七、API集成设计

### 预测端点
```
POST /api/v1/ml/predict
Body: { "stock_code": "600000", "model": "direction" }
Response: {
    "stock_code": "600000",
    "next_day_prob": 0.63,        # 次日上涨概率
    "next_5d_prob": 0.58,
    "next_20d_prob": 0.51,
    "risk_score": 0.32,
    "top_features": [
        {"name": "rsi_6", "importance": 0.12},
        {"name": "macd_histogram_change", "importance": 0.09},
    ],
    "signal": "BUY",              # ML增强信号
    "confidence": 0.63,
}
```

### 与现有AIScoreEngine集成
```python
# 在 AIRecommendation 中增加 ML 预测结果
ml_result = ml_predictor.predict(stock_code)
recommendation.ml_prediction = ml_result
recommendation.overall_score = (
    recommendation.overall_score * 0.7 +
    ml_result["next_day_prob"] * 30  # ML预测占30%权重
)
```
