# 竹林司马监控与日志系统设计

## 🚨 概述

为企业级应用提供完整的监控、告警和日志管理解决方案，确保系统高可用性、可观察性和故障快速恢复能力。

## 🎯 设计目标

### **核心目标**
1. **实时监控**: 秒级系统状态监控和告警
2. **全面可观察性**: 日志、指标、追踪三位一体
3. **快速故障诊断**: 分钟级问题定位和解决
4. **性能优化**: 实时性能分析和瓶颈识别

### **SLA承诺**
- 系统可用性: 99.95%
- 告警响应时间: <1分钟
- 故障诊断时间: <5分钟
- 数据保留: 日志30天，指标1年

### **技术栈选择**
- **指标监控**: Prometheus + Grafana
- **日志管理**: ELK Stack (Elasticsearch, Logstash, Kibana) 或 Loki
- **分布式追踪**: Jaeger 或 OpenTelemetry
- **告警管理**: Alertmanager
- **APM工具**: DataDog / New Relic (可选)

### **结构化日志格式**

```python
import structlog

logger = structlog.get_logger()

# 示例日志记录
def log_api_request(request: Request, response: Response, duration: float):
    logger.info(
        "api_request_completed",
        method=request.method,
        path=request.url.path,
        status_code=response.status_code,
        duration_ms=duration * 1000,
        user_agent=request.headers.get("user-agent"),
        user_id=request.user.id if hasattr(request, 'user') else None,
        request_id=request.state.request_id,
        correlation_id=request.headers.get("x-correlation-id"),
    )

def log_error(error: Exception, context: Dict = None):
    logger.error(
        "application_error",
        error_type=type(error).__name__,
        error_message=str(error),
        stack_trace=traceback.format_exc(),
        context=context,
    )
```

        labels:
          severity: warning
          service: api
        annotations:
          summary: "高延迟检测到"
          description: |
            API 95分位延迟超过1秒 (当前值: {{ $value }}s)
      
      - alert: ServiceDown
        expr: up == 0
        for: 1m
        labels:
          severity: critical
        annotations:
          summary: "服务下线"
          description: "{{ $labels.job }} 服务不可用"

  - name: database_alerts

### **Trace上下文传播**

```python
# 传播Trace ID到日志
import structlog
from opentelemetry import trace

def configure_structlog():
    structlog.configure(
        processors=[
            structlog.processors.TimeStamper(fmt="iso"),
            structlog.stdlib.add_logger_name,
            structlog.stdlib.add_log_level,
            structlog.stdlib.PositionalArgumentsFormatter(),
            # 添加Trace ID
            structlog.processors.StackInfoRenderer(),
            structlog.processors.format_exc_info,
            add_trace_context,  # 自定义处理器
            structlog.processors.JSONRenderer()
        ],
        context_class=dict,
        logger_factory=structlog.stdlib.LoggerFactory(),
        cache_logger_on_first_use=True,
    )

def add_trace_context(_, __, event_dict):
    """添加追踪上下文到日志"""
    span = trace.get_current_span()
    if span and span.is_recording():
        context = span.get_span_context()
        event_dict["trace_id"] = format(context.trace_id, "032x")
        event_dict["span_id"] = format(context.span_id, "016x")
    return event_dict
```
